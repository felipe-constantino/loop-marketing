# Biblioteca Tática — Ampliar (fase Amplify)

> Recursos táticos do **ampliar-agent**. NÃO são comandos invocáveis pelo usuário — são arquivos de aprofundamento que o agente lê **sob demanda** quando precisa produzir um entregável específico de canal/distribuição/amplificação.
>
> **Como usar (instrução para o agente):**
> 1. Identifique a necessidade tática durante o método base.
> 2. Escolha **UM** arquivo abaixo pelo "use quando".
> 3. Leia `biblioteca/Ampliar/<arquivo>.md` e use as instruções dele como **insumo**.
> 4. **O resultado volta SEMPRE pelo contrato de saída do ampliar-agent**: limite global de touchpoints, papel diferenciado por canal, classificação baseada em dados, grau de confiança e handoff. O prompt da biblioteca gera matéria-prima; a decisão continua sendo do agente.
>
> Regra: no máximo 1-2 arquivos por ciclo. **Não amplifique o que não foi validado** (erro nº 5 do agente). Se nenhum "use quando" casa, **não force**.

## Índice

- **estrategista-de-diversificacao-do-canal.md** — use quando: precisa identificar novos canais de alto potencial onde o público está.
- **diversificacao-de-midia-paga.md** — use quando: o foco é expandir/diversificar mídia paga com teste de eficiência.
- **orquestrador-de-experiencia-omnicanal.md** — use quando: precisa coordenar todos os touchpoints numa experiência coerente (Passo 4 do agente).
- **otimizador-de-atribuicao-cruzada-de-canal.md** — use quando: a atribuição é último-clique e precisa de modelo multi-touch.
- **mecanismo-de-amplificacao-de-conteudo.md** — use quando: precisa de framework de repurposing e distribuição de conteúdo que funciona.
- **estrategia-de-conteudo-entre-plataformas.md** — use quando: precisa adaptar 1 conteúdo central para várias plataformas mantendo a marca.
- **rede-de-distribuicao-de-conteudo.md** — use quando: precisa estruturar distribuição em mídia própria/ganha/paga.
- **otimizador-de-visibilidade-do-ai-search.md** — use quando: o objetivo é ser encontrado/citado por mecanismos de IA (eixo central do Loop Marketing na era da IA).
- **estrategia-viral-em-todas-as-plataformas.md** — use quando: a meta é amplificação viral com integridade de marca.
- **amplificador-de-conteudo-de-video.md** — use quando: o canal prioritário é vídeo.
- **estrategia-de-podcast-e-audio.md** — use quando: o canal prioritário é podcast/áudio.
- **construtor-de-rede-influenciadora.md** — use quando: precisa construir rede de influenciadores.
- **estrategista-de-parceria-creator.md** — use quando: a aposta é parceria com criadores de conteúdo.
- **distribuicao-voltada-para-a-comunidade.md** — use quando: o público está em comunidades e a entrada é via contribuição autêntica.
- **identificador-de-parceiro-estrategico.md** — use quando: precisa identificar e avaliar parceiros estratégicos.
- **mecanismo-de-co-marketing-da-parceria.md** — use quando: já tem parceiro e precisa estruturar co-marketing.
- **mecanismo-de-referencias-do-cliente.md** — use quando: precisa montar programa de referral/indicações (Passo 5 do agente).
- **amplificador-de-defesa-do-cliente.md** — use quando: precisa transformar clientes em defensores/advocacy (Passo 5 do agente).
- **multiplicador-de-conteudo-gerado-pelo-usuario.md** — use quando: a alavanca é UGC.
- **amplificacao-da-lideranca-do-pensamento.md** — use quando: precisa distribuir/amplificar thought leadership já definido pelo Verbalizar.
- **evento-ao-vivo-e-amplificador-de-webinar.md** — use quando: o canal é webinar/evento ao vivo.
- **amplificador-de-eventos-e-conferencias.md** — use quando: precisa extrair valor de participação em conferências/eventos do setor.
- **amplificador-de-marketing-de-eventos.md** — use quando: o foco é planejar marketing de eventos próprios.
- **amplificador-de-marketing-local-e-regional.md** — use quando: a estratégia é presença local/regional.
- **premios-da-industria-e-amplificador-de-reconhecimento.md** — use quando: a alavanca é prêmios/reconhecimento do setor.
