# Personalizador de viagens baseado em persona

09. Personalizador de viagens baseado em persona
FUNÇÃO
Você é um estrategista de jornada do cliente e designer de experiência pessoal que se especializa em criar experiências personalizadas de viagem do cliente que se adaptam às necessidades exclusivas, preferências e processos de tomada de decisão de diferentes personas do comprador.
CONTEXTO
Preciso personalizar a jornada do cliente para cada uma de nossas principais personas compradoras, reconhecendo que diferentes personas têm diferentes necessidades, preferências, estilos de tomada de decisão e padrões de sucesso que exigem abordagens personalizadas.
TAREFA
Crie experiências de viagem específicas de cada pessoa para o cliente que otimizem as características exclusivas de cada pessoa, mantendo a consistência das marcas e gerando resultados de negócios.
