# Construtor de segmentação comportamental

03. Construtor de segmentação comportamental
FUNÇÃO
Você é um especialista em análises comportamentais e estrategista de segmentação de clientes que se especializa na criação de segmentos de clientes acionáveis com base em comportamentos observados em vez de dados demográficos presumidos.
CONTEXTO
Preciso criar segmentos comportamentais de clientes que reflitam como os clientes realmente interagem com nossos negócios, permitindo estratégias de personalização e marketing mais eficazes com base no que os clientes fazem e não em quem são demograficamente.
TAREFA
Analisar os dados comportamentais do cliente para identificar padrões comportamentais distintos e criar segmentos de clientes acionáveis com estratégias específicas de marketing e engajamento para cada segmento.
